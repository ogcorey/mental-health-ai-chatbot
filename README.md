# Mental Health AI Chatbot

## Overview

This project focuses on building a sentiment-aware AI chatbot designed to provide supportive conversations for users experiencing stress, anxiety, or emotional challenges. The system is being developed with a strong emphasis on personalization, ethical AI usage, and real-world applicability.

## Objectives

* Develop a conversational AI chatbot with emotional awareness
* Implement sentiment and crisis detection mechanisms
* Enable optional daily check-ins for users
* Support multi-platform integration (WhatsApp, Web, Mobile)

## Current Status

This project is currently in the research and design phase. Initial work focuses on understanding user needs, defining system architecture, and exploring AI model capabilities.

## Core Features (Planned)

* User personalization (custom bot naming)
* Sentiment analysis and classification
* Daily check-in system
* Crisis detection and safe response handling
* Conversation memory (context awareness)

## Tech Stack (Planned)

* Backend: Python (FastAPI) or Node.js
* AI Models: Local LLMs (Ollama) / API-based models
* Database: MongoDB / PostgreSQL
* Integration: WhatsApp API (Twilio), Web interface

## Repository Structure

* /research → Problem analysis, datasets, and findings
* /design → System architecture and feature planning
* /backend → API and chatbot logic (in progress)
* /prompts → Prompt engineering and response templates
* /docs → Documentation and notes

## Ethical Considerations

This system is not a replacement for professional mental health care. It is designed to provide supportive interactions and guide users toward appropriate help when necessary.
